# <img src="public/icons/icon_48.png" width="45" align="left"> Flomo Input

一个始终至于浏览器最前面的 flomo 输入窗口， 简化记笔记的流程使你有想法或灵感能及时记录下来。

## 特点

- 输入窗口处于浏览器的最前面
- 会判断是否有浏览器操作行为而选择什么时候弹出窗口
